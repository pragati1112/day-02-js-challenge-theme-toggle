# 🚀 Day 02 — Theme Toggle (JavaScript Challenge)

This project is part of my **21 Days JavaScript Challenge** where I build real-world frontend projects using HTML, CSS, and JavaScript.

---

## 📌 Project Overview

A simple and interactive **Theme Toggle App** that allows users to switch between **Light Mode ☀️ and Dark Mode 🌙**.  
The selected theme is saved using **localStorage**, so it remains even after page reload.

---

## 🎯 Goal

- Practice DOM manipulation  
- Understand class toggling  
- Learn how to use localStorage  
- Build user-friendly UI  

---

## 🧩 Features

- 🌙 Dark Mode  
- ☀️ Light Mode  
- 🔁 Toggle Button  
- 💾 Theme saved in localStorage  
- 🔄 Persistent theme after reload  

---

## 🧠 Flow (Planning Before Coding)

1. Check if theme is saved in localStorage  
2. Apply saved theme on page load  
3. Add event listener to toggle button  
4. Toggle "dark" class on body  
5. Save selected theme in localStorage  
6. Update UI instantly
